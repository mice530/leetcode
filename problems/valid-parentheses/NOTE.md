# 题目
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。
有效字符串需满足：
左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:
输入: "()"
输出: true

示例 2:
输入: "()[]{}"
输出: true

示例 3:
输入: "(]"
输出: false

示例 4:
输入: "([)]"
输出: false

示例 5:
输入: "{[]}"
输出: true

# 原理
空间优先：遍历一次计数，遍历第二次取到对应的节点
时间优先：因为节点数有限，可以考虑转存到数组，省去第二次遍历时间

# 复杂度分析
时间O(N)：链表线性遍历
空间O(1)：少数变量开销
